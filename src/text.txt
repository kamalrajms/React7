## react intro

    What is React?

    1) React is a JavaScript library (not a framework) developed by Facebook (Meta) for building user interfaces (UIs), especially Single Page Applications (SPAs).

    2) It helps developers build reusable UI components that update efficiently when data changes.

    Instead of reloading a whole webpage when something changes, React updates only the parts of the UI that need to change.

    -------
    Why do we need React? 

	It makes UI development faster, cleaner, and more efficient.

	Provides a better user experience with smooth updates without page reloads.

## useState() and handleing forms

    useState is a react hook that allows you to add state (data that can 
    changes over time) to a functional component.

    normally , in js functions ,variables disappear after the function runs,but
    with useState ,  react remember values between re-renders.

    syntex: 
        const[state,setState]=useState(initiate_state)

        state---the current value of the state variable.
        setState--- a function that updates the state.
        initiate_state-- the starting value  of the state.

## Routing in react && nested routing

    Routing means navigating between different pages in a web applaciation
    without refreshing the browser.

    In Tradition webSites ,Clicking a link reloades the whole Applications.

    In react (a single page application) routing lets us changes the view without reloading.

                Better performance (no full page reloads)
                Smooth user experience
                Control over navigation without server request.

    commend: npm install react-router-dom

    Structure:

            <BrouserRouter>
            <Routes>
            <Route path="address path" element={<component1/>}/>
            <Route path="address path" element={<component2/>}/>
            <Route path="address path" element={<component3/>}/>
            <Routes path="address path" element={<component4/>}/>
            </Routes>
            </BrouserRouter>

            BrouserRouter
                it is the main wrapper component for   routing in React app.
                It enables navigating using the browser URL's  (history)

            Routes
                It contains all the <Route/> component
                It tells react Router which component to show for which path
            
            Route

                Defines a specific url path and which component should lood  for
                the path.
